
## Data
Download [CelebA](https://mmlab.ie.cuhk.edu.hk/projects/CelebA.html) Align&Cropped Images.

## Use
1. 更改load_celebA.py文件中get_dataloader函数的数据路径
2. 更改main_vae.py中函数生成图片的路径
3. 直接跑main_vae即可完成训练和生成
